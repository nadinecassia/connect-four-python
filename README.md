# connect-four-python
Jogo Connect Four desenvolvido em Python com foco em POO - 1ª Fase de Sistemas de Informação (UFSC).
Este projeto foi desenvolvido como parte prática dos estudos de **Sistemas de Informação na UFSC**, com foco na aplicação de conceitos fundamentais de POO.

## Sobre o Projeto
O jogo foi desenvolvido em **Python** e implementa a lógica clássica do Connect Four. O sistema permite os modos Humano x Humano e Humano x Computador (PvE). O projeto prioriza a organização do código e a integridade dos dados através de pilares da Orientação a Objetos aprendidos na primeira fase do curso.
## Conceitos Aplicados (Habilidades Técnicas)
De acordo com a base acadêmica da UFSC, este projeto exercita:

* **Encapsulamento**: Uso de atributos privados (`self.__tab`) para garantir que o estado do tabuleiro seja alterado apenas por métodos internos da classe `Tabuleiro`, protegendo a integridade dos dados.
* **Modelagem de Objetos**: Separação clara de responsabilidades entre as classes `Tabuleiro` (Lógica e Dados) e `Jogo` (Controle e Fluxo).
* **Lógica de Matrizes**: Implementação de algoritmos de busca para verificação de vitória em eixos horizontais, verticais e diagonais.
* **Tratamento de Exceções**: Validação de entradas do usuário para garantir a robustez do sistema, evitando que jogadas inválidas interrompam a execução.
